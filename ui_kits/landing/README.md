# Landing page UI kit

Recreation of Inner Circle's marketing surface — the public-facing landing that the chat brief asked us to redesign.

## Components
- `Hero.jsx` — full-bleed dark hero with serif headline, mascot, primary CTA
- `OrbitGraphic.jsx` — the trust illustration: you at center, providers orbiting
- `HowItWorks.jsx` — three-step explanation (MyCircle / MyEcosystem / MyFriends)
- `CompareStrip.jsx` — "stars from strangers" vs "people you actually know"
- `ProviderCTA.jsx` — secondary funnel for local providers
- `FAQ.jsx` — accordion FAQ
- `Footer.jsx`

## Open in `index.html` to see the full assembled page.
