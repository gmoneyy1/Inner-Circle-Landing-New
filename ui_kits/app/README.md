# App UI kit

Recreation of the Inner Circle product app — four-tab experience.

## Components
- `AppShell.jsx` — glassy header + bottom tab nav + view router
- `MyCircleView.jsx` — orbit visualization of the providers you trust
- `FriendsView.jsx` — vertical wheel of friends' circles
- `EcosystemView.jsx` — town directory by category
- `ProviderDetail.jsx` — provider detail sheet

Open `index.html` to click through the tabs.
